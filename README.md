# AnchorSlide.js
a simple jQuery plugin to animate anchor tags.


`$("#your-element").AnchorThis();`
### Settings
speed(in Milliseconds):

```
$("#your-element").AnchorThis({
  speed: 100
});
```
target:

```
<<<<<<< HEAD
$("#your-element").AnchorThis({
  target: a.only-some-a-tags
});

```

_can be combined_
```
$("#your-element").AnchorThis({
  speed: 100,
  target: a.only-some-a-tags
});
```
